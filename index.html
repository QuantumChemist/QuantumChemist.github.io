About Me

Hello, I am Dr. Christina Ertural, a Python and C++ software developer specialized in deploying scientific applications.

My focus lies on employing automation tools for scientific workflows for machine learning and chemical bonding analysis in materials informatics.

I also have several smaller projects using C++, Python and TS for several different applications. Check out my GitHub profile to learn more.
